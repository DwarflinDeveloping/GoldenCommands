# Was ist "GoldenCommands"?
GoldenCommands ist ein Tool für GoldenArmy-Nutzer mit dessen Hilfe die vollen Funktionen von Ingame-Kommandos schnell herraus gefunden werden kann.

# Was ist noch geplant?
